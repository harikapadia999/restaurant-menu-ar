# 🍽️ Restaurant AR Menu

> Interactive 3D Food Visualization & Virtual Menu Experience

[![Live Demo](https://img.shields.io/badge/demo-live-success)](https://harikapadia999.github.io/restaurant-menu-ar/)
[![A-Frame](https://img.shields.io/badge/A--Frame-1.5.0-blue)](https://aframe.io/)
[![WebXR](https://img.shields.io/badge/WebXR-Compatible-orange)](https://immersiveweb.dev/)

## 🌟 Features

- **3D Food Models** - Realistic 3D visualization of dishes
- **Interactive Menu** - Browse categories and items
- **Detailed Information** - Ingredients, nutrition, and pricing
- **Shopping Cart** - Add items and track orders
- **Customization Options** - Modify dishes to preferences
- **Ratings & Reviews** - See dish ratings
- **Rotating Display** - 360° view of dishes
- **VR Support** - Immersive dining experience

## 🚀 Quick Start

Simply open the live demo link in any modern web browser. No installation required!

### For VR Experience
1. Open the demo on a VR-compatible device
2. Navigate through the menu
3. Click dishes to view in 3D
4. Add to cart and customize

## 💻 Technology Stack

- **A-Frame 1.5.0** - WebVR framework
- **A-Frame Extras** - Additional components
- **Particle System** - Visual effects
- **HTML5/CSS3** - Modern web standards
- **JavaScript ES6** - Interactive functionality

## 🔧 Customization

To add your own menu:
1. Create 3D food models in the scene
2. Update the `menuData` object with your items
3. Add dish information and pricing
4. Customize colors and branding
